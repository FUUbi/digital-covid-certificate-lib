Digital Covid Certificate Lib (DccLib) - Js Demo:

This is a simple demo on how to use the DccLib library with a JavaScript project.

Please upload or scan a valid covid certificate QrCode.

Set your trust anchor by providing a root certificate. This demo uses the Swiss Government's root certificate "CA II" per default.


You can also provide a custom list of DCC keys as a Json Web Token (JWT). Make sure that the JWT header and payload follow the same structure as provides by the Swiss update list. more info.
